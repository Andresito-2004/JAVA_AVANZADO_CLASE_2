En este proyecto realizamos un formulario en el cual utilizamos boostrapp para darle estilos, hicimos el formulario de ingreso y un formulario de registro con campos de validacion en la cual podemos valiidar que se ingresen mmayusculas, minusculas, minimo y mayor, tambien utilizamos la sentencia "REQUIRE" con la cual se le solicitara al usuario que debe ingresar los datos obligatoriamente, el formulario esta conformado por un Header,Body y un footer, en este caso este sistema se realizo tambien un a conexion a una base de datos y se creo la base de datos en MySQL
